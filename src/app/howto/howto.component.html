Removing Users from Groups

If you remove a user from a group during a demo session, there is NO need to add the users back into the group. A Workflow runs every night to add the users back into the groups.

  1. Open the Org Admin console

    • Go to https://mortpanda.github.io/jpn-corp-ae-app-launcher/


  2. In the Admin console go to "Groups"

    • From the navigation menu go to "Directory" then "Groups"


  3. Group to remove the users from

    • The applications displayed in the Portal App is to be controlled using 3 x Okta groups assigned to the application.

    • Click on "Assign People" button.
    • Select the "-" next to the user to remove.
